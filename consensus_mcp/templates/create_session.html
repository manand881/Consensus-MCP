How It Works

  1. Create the session with a topic and goals
  2. LLM adds agents using the add_agent tool with appropriate types:
    • scientist - Tests hypotheses with code/experiments
    • perspective - Provides domain expertise and challenges
    • synthesizer - Tracks convergence and identifies agreements
    • moderator - Manages flow and enforces structure
  3. Debate begins - Agents share reasoning, challenge claims, run experiments
  4. Consensus emerges - Synthesizer identifies agreements, moderator guides toward conclusion
